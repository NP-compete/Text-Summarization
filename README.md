# Text-Summarization
Returns summary of text

### To run:
1. Git clone this repo
```
git clone https://github.com/NP-compete/Text-Summarization
```
2. cd to the Directory
```
cd Text-Summarization
```
3. Open Summarize.py
```
vi Summarize.py
```
4. Change content in variable  ```text```
5. Close the file
6. Run the code
```
python Summarize.y
```
